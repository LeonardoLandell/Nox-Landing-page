# NOX Landing Page

**Versão Final Para Portfólio Digital**

O **NOX** é um projeto desenvolvido em **Next.js** com foco em animações modernas e experiência imersiva.  
Ele representa a versão final da minha landing page/portfólio, criada para demonstrar habilidades em:

- React + Next.js  
- GSAP para animações avançadas  
- Lenis para smooth scrolling  
- Swiper para carrosséis interativos  
- Deploy otimizado em Vercel  

## 🎯 Objetivo
Servir como demonstração pública das minhas habilidades em desenvolvimento front-end e design de interfaces modernas, além de ser um projeto de referência para futuros trabalhos.

## 🚀 Deploy
- **Backup e versionamento:** GitHub  
- **Demonstração online:** Vercel *(URL será gerada após o deploy)*
